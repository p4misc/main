Perforce Server Log Analyzer - ローカル Docker バージョン

Helix Coreサーバのパフォーマンスに関する問題の診断方法

Perforce Server Log Analyzer (PSLA)にログファイルをアップロードします。
ログには、Helix Coreサーバのパフォーマンスに影響を与えた可能性があるコマンドの詳細が記述されています。
ログデータは、Dockerコンテナ内のSQLite3データベースに格納されます。

PSLAは log2sql.py ツールのWeb UI版です。
log2sql.py自体は、カスタムレポートを作る機能を備えていますが、Pythonと関連パッケージをインストールする必要があります。また、DockerのインストールやSQLの知識も必要になります。
PSLAは、Helix Coreのログファイルをアップロードして結果を解析するだけで簡単に利用できます。
